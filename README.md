# Project Product Portfolio Dashboard

## Overview

This project demonstrates the end-to-end creation of a project portfolio monitoring solution using Dataiku DSS and Power BI.

The objective was to monitor project performance, budget utilization, risk exposure and portfolio health through interactive dashboards.

## Tools Used

- Dataiku DSS
- Power BI
- DAX
- CSV Dataset

## Data Preparation

Data was cleaned and transformed in Dataiku DSS.

Additional business metrics were created:

- Risk Level
- Budget Variance
- Budget Status
- Cost Utilization %
- Projects Behind

## Dashboard KPIs

- Total Projects
- Projects Behind
- Total Budget
- Total Actual Cost
- High Risk Projects
- Average Progress

## Visualizations

- Risk Distribution (Donut Chart)
- Priority Analysis
- Budget vs Actual Cost
- Detailed Project Monitoring Table
- Interactive Filters (Project Type and Location)

## Business Insights

- 87% of projects are behind schedule.
- 45 projects are classified as High Risk.
- Average portfolio progress is 63.6%.
- Budget consumption remains below the total allocated budget.
